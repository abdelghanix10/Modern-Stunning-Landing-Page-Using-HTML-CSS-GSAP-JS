# Modern Stunning Landing Page Using HTML, CSS, GSAP & JS

A sleek and modern landing page designed to showcase design inspirations and news. Built with HTML, CSS, and enhanced with GSAP animations for smooth, engaging user interactions.

## Features

- **Responsive Design**: Optimized for various screen sizes and devices.
- **Smooth Animations**: Powered by GSAP (GreenSock Animation Platform) for professional-grade animations.
- **Modern UI**: Clean and stunning visual design with typography and layout.
- **Interactive Elements**: Hover effects, transitions, and animated cards.
- **Social Integration**: Links to social media platforms like Dribbble, Behance, Instagram, and Twitter.

## Technologies Used

- **HTML5**: Structure and content.
- **CSS3**: Styling and layout.
- **GSAP**: JavaScript animation library for smooth transitions.
- **Remixicon**: Icon library for modern icons.

## Preview

The landing page features:

- A top navigation bar with logo, menu, search, and hire button.
- Animated title "NEWS" with letter-by-letter reveal.
- Inspirational text and a yellow background animation.
- Three cards highlighting design topics: Marketing Materials, Rebranding, and Web Design Trends.

## How to View

1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. Enjoy the animations and interactions!
